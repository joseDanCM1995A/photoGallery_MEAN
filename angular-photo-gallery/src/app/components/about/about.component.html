Photo Gallery App

Aplicación web que te permite almacenar momentos de nuestra vida para nunca olvidarnos de ellos.

Aplicación desarrollada con el Stack MEAN (MongoDB, Express, Angular, Nodejs). Es una aplicacióm fullstack utilizando TypeScript; tanto para el Frontend como para el Backend. Dentro del desarrollo también se utilizó el framework de CSS llamado materializeCSS, para poder tener un estilo más llamativo visualmente.

Aplicación desarrollada por José Daniel Contreras Medina
